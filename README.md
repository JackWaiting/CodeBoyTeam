# CodeBoy版开源中国
  初次看到这个开源项目源码的时候，感觉很不错，里面有许多东西和小知识点供大家学习和讨论，通过对此项目的学习和完善，对本项目做如下总结，如果你想学习这些内容的话，可以下载看看，已经尽力加啦好多注释吧，本人后期也会利用业余时间，继续完善这个项目。
  知识点总结如下：
	1、Fragment+ViewPage+ActionBar(android-support-v7兼用包)。
	2、第三方接口引用与第三方登陆。
	3、WebService运用，与开源中国数据对接。
	4、上拉刷新下拉加载。
	5、音频、视频的上传。
	6、录音功能的完善和声波感应。
	7、聊天表情的运用。
	8、上传与下载。
	9、评论与收藏。
	10、后台推送技术。
  	
   暂时把知识点分为以上部门，具体内容并不只是这些，本人上传这个项目也只是想与大家分享一下这些内容，毕竟在我们的开发工作中，这些内容也是非常常见的，2年工作内的小伙伴们还是很有必要学习学习，当然我上传的目的是希望通过github这个平台能吸引更多的人加入这个项目，能一起完善这个项目，这个项目还是有一定的意义，已经有很多程序员在里面提问与回答问题。
   
	
   哈哈，其实，每天利用业余时间去一点点地编写，积累下来的效果还是不错的。希望大家可以完善这个项目，也期待开源中国越来越好。
